# Sample Images

Esta carpeta debe contener imágenes de muestra para pruebas rápidas.

**NOTA**: Las imágenes de muestra NO están incluidas en el repositorio Git para mantener el tamaño del repo pequeño.

## Cómo obtener imágenes de muestra

Puedes copiar algunas imágenes del dataset SKU-110K aquí para pruebas rápidas:

```bash
# Desde la carpeta principal del dataset
cp data/raw/SKU110K_fixed/images/test_*.jpg data/raw/sample/
```

O usar tus propias imágenes de anaqueles de tiendas.

## Uso

Las imágenes en esta carpeta se usan para:
- Pruebas rápidas del pipeline
- Validación de cambios sin procesar todo el dataset
- Ejemplos de demostración
